# Liuaj's Blog

> 学会积累，厚积薄发

个人技术博客，基于 [Jekyll](https://jekyllrb.com/) 构建，部署于 [GitHub Pages](https://pages.github.com/)。

## 📖 博客简介

这是 Liuaj 的个人技术博客，记录学习过程中的技术积累与思考。博客涵盖后端开发、数据库、操作系统、开发工具等多个技术领域。

**博客地址**: [https://Liu-aj.github.io](https://Liu-aj.github.io)

## 📂 项目结构

```
├── _posts/          # 博客文章 (Markdown 格式)
├── _layouts/       # 页面布局模板
├── _includes/       # 页面组件 (header, footer 等)
├── _data/          # 数据文件
├── _sass/          # Sass 样式文件
├── styles/         # CSS、JS、图片等静态资源
├── about/          # 关于页面
├── categories/     # 分类页面
├── tag/            # 标签页面
├── _config.yml     # Jekyll 配置文件
└── index.html      # 首页
```

## 📝 文章分类

| 分类 | 描述 |
|------|------|
| **Java** | Java 开发、线程并发、JDK 配置等 |
| **DB** | 数据库相关 (MySQL、MongoDB、MyBatis) |
| **OS** | 操作系统 (Linux、Windows、MacOS) |
| **加密** | 安全加密相关 (MD5 等) |
| **Other** | 其他技术内容 (Git、Maven、Chrome 等) |

## 🛠️ 技术栈

- **静态站点生成器**: Jekyll
- **Markdown 解析**: Kramdown
- **前端框架**: Bootstrap
- **部署平台**: GitHub Pages

## 🚀 本地运行

```bash
# 安装依赖
bundle install

# 启动本地服务器
bundle exec jekyll serve

# 访问 http://localhost:4000
```
